# 玩法
1. 收集
从用户中收集消息。
2. 推送
将收集到的消息推给用户。
3. 平衡
用户将改变自身对其它用户的态度。
4. 生产
用户在改变态度的过程中产生新消息。

# 机制
G(p)(o,x)：p视角下o与x的关系。

- 推送消息
消息携带的信息：a对b1的态度是v1, a对b2的态度是v2...
推送消息：让用户知道消息携带的所有内容
p知道o对x的态度是v：G(p)(o,x)=v

- 平衡
用户p的不平衡度S(G)=对于所有o，对于所有x，(GpxGxo-Gpo)^2之和
G:n阶对称方阵
Gab:p视角下a对b的态度
Gaa=0
D(o)=对于所有x，2(Gox+1)Gpo-4GoxGpx之和。
D(o)表示S(G)对Gpo的偏导。
记o使|D(o)|取得最大值，用户的平衡行为即Gpo+=d/D(o)，d为用户的激进度。
